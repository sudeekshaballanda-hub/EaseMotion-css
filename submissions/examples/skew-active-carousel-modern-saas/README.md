# CSS Skew-Active Carousel for Modern SaaS Layouts

**Issue:** #50494

## Overview

A responsive **pure CSS Skew-Active Carousel** for Modern SaaS landing pages featuring smooth skew animations and zero JavaScript.

## Features

- Pure HTML & CSS
- Responsive
- Keyboard accessible
- Smooth skew + scale animation
- CSS Custom Properties
- Lightweight
- Zero JavaScript

## Usage

```html
<div class="carousel">
    <div class="card active">Analytics</div>
    <div class="card">Automation</div>
    <div class="card">Security</div>
</div>
```

## Customization

```css
:root{
    --duration:.45s;
    --scale:1.08;
    --skew:-7deg;
    --primary:#4f46e5;
}
```
