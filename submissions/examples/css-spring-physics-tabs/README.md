# CSS Spring Physics Tabs

A pure CSS animated tab component with a smooth spring physics interaction for modern minimalist interfaces.

## ✨ Features

- Pure CSS
- Spring-style hover animation
- Keyboard accessible
- Responsive layout
- Custom CSS variables
- No JavaScript required

## Usage

```html
<div class="tabs">
  <button class="tab active">Home</button>
  <button class="tab">Projects</button>
</div>
```

## Custom Properties

```css
:root{
  --spring-duration:0.45s;
  --spring-easing:cubic-bezier(.2,1.4,.4,1);
  --spring-scale:1.08;
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component demonstrates a reusable spring-style interaction using only CSS transforms and transitions while remaining lightweight, responsive, and easy to customize.
