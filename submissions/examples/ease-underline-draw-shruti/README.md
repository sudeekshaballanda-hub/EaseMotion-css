# ease-underline-draw

Animated underline component for links built with pure CSS.

## Features

- Left-to-right underline animation
- Center-out underline animation
- Right-to-left underline animation
- Persistent active underline
- Pure CSS
- Keyboard accessible
- Responsive
- Customizable via CSS variables

## CSS Variables

```css
--ease-underline-color
--ease-underline-height
--ease-underline-offset
--ease-underline-duration
--ease-underline-easing
```
