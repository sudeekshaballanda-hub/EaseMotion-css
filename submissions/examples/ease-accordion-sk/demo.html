What is EaseMotion CSS?

EaseMotion CSS is a human-readable, animation-first CSS framework with zero dependencies. It provides utility classes, animations, and components that are self-describing — the class name is the documentation.

No build step required. Just drop the CSS file into your project and start using ease-fade-in, ease-hover-grow, or ease-btn directly.

How do I install it?

You have several options:

  • CDN: <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.min.css">
  • npm: npm install easemotion-css
  • Download: Grab easemotion.min.css from the GitHub releases page
Does it require JavaScript?

No. EaseMotion CSS is pure CSS. All animations, utilities, and components work without JavaScript. The only optional JS files (core/reveal.js, core/tabs.js) provide progressive enhancements like scroll-triggered reveals and keyboard navigation.

Is this framework production-ready?

Yes. EaseMotion CSS v1.0 has been tested across all modern browsers and includes accessibility defaults, dark mode support, and responsive utilities.

Can I customize the design tokens?

Absolutely. All design values are stored as CSS custom properties in core/variables.css. Override any token — colors, spacing, typography, shadows — in your own stylesheet and the entire framework adapts.

What browsers are supported?

Chrome, Firefox, Safari, and Edge — all modern versions. EaseMotion CSS uses widely supported CSS features with progressive enhancement for newer APIs.

Exclusive Group (JS-enhanced)

Single-select behavior where only one item can be open at a time. Uses a light JavaScript enhancement for browsers that don't support the name attribute on <details>.

Overview

This accordion variant uses JavaScript to enforce single-select behavior. When you open one item, any other open item automatically closes.

Behavior

Clicking a new item closes the previously open one. This pattern is common for FAQs, settings panels, and navigation menus where you want focused navigation.

Graceful Degradation

If JavaScript is disabled, the accordion gracefully falls back to standard multi-open behavior. All content remains accessible.
